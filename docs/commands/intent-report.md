# /intent-report

> Generate human-readable reports from Intent files

## Usage

```bash
/intent-report
/intent-report --format html
/intent-report --format pdf
/intent-report --output docs/
```

## Purpose

Converts technical Intent specifications into readable documents for stakeholders, team members, or project documentation.

## When to Use

- Sharing Intent with non-technical stakeholders
- Creating project documentation
- Generating API documentation
- Team onboarding materials

## Output Formats

| Format | Use Case |
|--------|----------|
| **markdown** | Default, for docs sites |
| **html** | Standalone web pages |
| **pdf** | Printable documents |

[API section formatted as documentation]

**Parameters:**
| Name | Type | Description |

**Example:**
\```
[from Examples section]

## Dependencies

[Dependencies section as table]

[Constraints section]

---
*Generated by IDD /intent-report*

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--format` | Output format (markdown/html/pdf) | markdown |
| `--output` | Output directory | Current directory |
| `--include-private` | Include draft sections | false |
| `--toc` | Generate table of contents | true |

## Multi-File Reports

For project-level reports:

```bash
/intent-report intent/ --format html
```

This generates a combined report with:
- Project overview from `intent/INTENT.md`
- Module sections from each module's Intent
- Dependency diagrams from architecture files

## Customization

### Report Templates

Place custom templates in `intent/.templates/`:

```
intent/
├── .templates/
│   ├── report-header.md
│   ├── report-footer.md
│   └── section-api.md
└── INTENT.md
```

### Excluding Sections

Add to Intent frontmatter:

```markdown
---
report:
  exclude:
    - internal-notes
    - draft-ideas
---
```

## Use Cases

### Stakeholder Documentation

```bash
/intent-report --format pdf --output stakeholder-docs/
```

### API Documentation Site

```bash
/intent-report intent/api/ --format html --output docs/api/
```

### Team Onboarding

```bash
/intent-report intent/ --format markdown --output wiki/
```
